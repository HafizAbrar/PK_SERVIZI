# Generate App Icon

Your app is already configured to use `outer logo.png` as the app icon.

## Run this command to generate the app icon:

```bash
flutter pub run flutter_launcher_icons
```

This will automatically:
- Generate Android app icons in all required sizes
- Generate iOS app icons in all required sizes
- Update AndroidManifest.xml and iOS configuration files

After running this command, rebuild your app and the outer logo will appear as the app icon on the mobile device's home screen.
